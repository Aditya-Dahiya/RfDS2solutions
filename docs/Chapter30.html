Let us try to create an interactive map in HTML below Figure 1:—̥

library(leaflet)
leaflet() |>
  setView(76.801175, 30.761403, zoom = 14) |> 
  addTiles() |>
  addMarkers(76.801175, 30.761403, 
             popup = "Haryana Civil Secretariat")  |>
  addMarkers(76.803773534,
             30.752910586,
